## Overview

Caption Studio Tools is a web-first HTML toolkit for caption-focused studio workflows. It is intended to be opened in a browser, giving you access to caption utilities without requiring a separate desktop app or a heavier installation process.

The project is delivered as a small static site, which keeps publishing and upkeep straightforward across different hosting environments. It works well for teams or solo users who need a compact browser tool that can be deployed directly from a repository with very little configuration.

## Installation

   `git clone https://github.com/mattx1988/caption-studio-web-tools.git
2. Open the project folder:
   `cd caption-tools-for-studio`
3. Serve or open the HTML entry point in a browser.

## Usage

Open the interface in a supported browser and follow the on-screen steps for caption-related work. Because the toolkit is built with HTML, the usual workflow is:

1. Load the site in a browser.
2. Go to the caption tool or screen you need.
3. Complete the available studio workflow steps.
4. Save or refresh as needed based on your hosting setup.

For hosted deployments, users can reach the site directly through the published URL. For local use, keep the repository files available through a simple static server.
